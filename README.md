
본 문서는 리눅스 환경에서 프로세스를 확인하고 제어하는 핵심 명령어인 `top`, `ps`, `jobs`, `kill`에 대해 정리한 과제입니다.

---


| 명령어 | 주요 기능 | 특징 |
| :--- | :--- | :--- |
| **`top`** | 실시간 시스템 리소스 및 프로세스 모니터링 | Windows의 '작업 관리자'와 유사, 실시간 갱신 |
| **`ps`** | 현재 실행 중인 프로세스의 스냅샷 출력 | 실행 시점의 프로세스 상태를 정적으로 확인 |
| **`jobs`** | 현재 셸 세션에서 관리되는 작업 목록 표시 | 백그라운드/포어그라운드 작업 상태 확인 |
| **`kill`** | 특정 프로세스에 시그널을 보내 종료 | PID 또는 작업 번호를 지정하여 프로세스 제어 |

---


시스템의 전반적인 상태와 프로세스 리소스(CPU, 메모리) 사용량을 **실시간으로 화면에 출력**합니다.

* **주요 옵션 및 단축키:**
  * `M`: 메모리 사용량 순으로 정렬
  * `P`: CPU 사용량 순으로 정렬 (기본값)
  * `q`: `top` 화면 빠져나가기
* **사용 예시:**
  ```bash
  top
